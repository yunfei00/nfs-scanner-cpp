# 运动串口测试说明

## 硬件连接

1. 将运动控制器通过 USB/串口连接到 Windows 主机。
2. 确认控制器供电正常，急停和限位状态允许运动。
3. 在设备管理器中确认串口号，例如 COM3、COM10。

## 串口参数

- 波特率：115200
- 数据位：8
- 校验：None
- 停止位：1
- 流控：None

## 支持命令

- `$H`：复位/回零
- `?`：位置查询
- `$I`：读取版本
- `$`：帮助命令
- `G1X..Y..Z..F..`：绝对坐标运动，命令必须包含 Feed

## 测试步骤

1. 取消勾选“模拟模式”。
2. 点击“刷新串口”，选择目标端口。
3. 点击“打开串口”。
4. 点击“读取版本”，确认日志中有控制器返回。
5. 点击“位置查询”，确认可以解析 `<Idle|MPos:...>` 状态。
6. 选择 1 mm 步距，点击 X+。
7. 点击 Y-。
8. 点击 Z+。
9. 点击“复位”，确认发送 `$H`。

## 常见问题

- 找不到串口：检查 USB 线、驱动和设备管理器中的端口。
- 串口被占用：关闭串口调试助手、上位机或其他占用该端口的软件。
- 发送命令无响应：确认波特率为 115200，控制器协议为 GRBL-like，命令结束符为换行。
- 坐标越界：当前限制为 X 0~200、Y -300~0、Z 0~10。
- 返回格式解析失败：确认位置返回包含 `MPos:x,y,z`，例如 `<Idle|MPos:110.000,-100.000,5.000|FS:0,0>`。
